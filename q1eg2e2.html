WHAT ARE JAVASCRIPT OPERATORS?

WHAT IS JAVASCRIPT?

Javascript is a coding language that can be used to create web pages. This is a software application that let's you do nearly everything.

WHAT ARE JAVASCRIPT OPERATORS?

- In JavaScript, an operator is a special symbol used to perform operations on operands (values and variables). Javascript has only a few types of operators such as arithmetic, comparison, logical and assignment.

ARITHMETIC OPERATORS

-Arithmetic operators are used to execute numerical operations on operands.

OPERATOR
DESCRIPTION
+
for adding
-
for subtracting
/
for dividing
*
for multiplying

- ADDITION OPERATOR

( x + y ) This combines the variables and also remainder of addition. This adds two or more numbers into one outcome. This adds the correct operand value to a variable and returns the variable's result.

- SUBTRACTION OPERATOR

( x - y ) this combines the variables and also the remainder of subtraction. (-=) for subtracting the right operand out from the left operand and transferring the result to the variables here on the left. This deducts numbers or takes out a certain group number.

- MULTIPLICATION OPERATOR

( x * y) this combines the variables and also the remainder of multiplication. it multiplies the variable by the right operand value and gives the outcome to the variable.

- DIVISION OPERATOR

( x / y ) This combines the variables and also a remainder of division. This is a procedure of dividing a number into equal parts and determining how many equal portions could be made.

HOW TO USE THE COMPARISON OPERATION

(Scenario: Using this operation will compare two operands and return a boolean value as true or false.)

- GREATER THAN

if the left operand is greater than the right operand, > analyzes to true.

- LESS THAN

If the left operand is less than the right operand, then evaluates to true.

- GREATER THAN OR EQUAL TO

If the left operand is greater than or equal to the right operand, >= returns true.

- LESS THAN OR EQUAL TO

If the left operand is smaller than or equal to the right operand, <= returns true.

- EQUAL TO

If the operands are equal, == returns true.

- NOT EQUAL TO

!= If the operands are not equal, = returns true.

-Logical Operators combine the three variables.

- LOGICAL 'AND'

is written as && and it identifies if two operands are non-zero or not. It is either null, undefined, false, 0 or ”” are considered as zero. It returns to 1 if they are the non-zero; otherwise, returns to 0.

- LOGICAL 'OR'

is written as | | and it identifies if any operands are non-zero or not. It is either null, undefined, false, 0 or ”” is considered as zero. It returns to 1 if any of them is non-zero; otherwise, it returns to 0.

- LOGICAL 'NOT'

is written as !, what it does is that it reverses the boolean result of the operand or condition, like !false returns true, and !true returns false.
